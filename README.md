# CryptoBot

This is a proof of concept, for a full featured trading bot try [YATB](https://github.com/lropero/yatb).

### Requires

- Node v13.12.0

### Installation

```sh
$ npm install
```

### Usage

Add Binance keys to `config.js`, then:

```sh
$ npm run start
$ curl http://localhost:7007/chart/BTCUSDT | json_pp
```
